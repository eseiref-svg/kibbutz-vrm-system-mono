##   驻专拽

```

```

##  转拽 专

```bash
# 转拽转 -server
cd naan-vrm-server
npm install

# 转拽转 -client
cd ../naan-vrm-client
npm install
```

##  专爪

```bash
# Server (Terminal 1)
cd naan-vrm-server
npm start  # 专抓 注 http://localhost:5000

# Client (Terminal 2)
cd naan-vrm-client
npm start  # 专抓 注 http://localhost:3000
```
